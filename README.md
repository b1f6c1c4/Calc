# Calc
A simple calculator written in Verilog

# Keys
|        |  V[3]  |  V[2]  |  V[1]  |  V[0]  |
|:------:|:------:|:------:|:------:|:------:|
|  H[3]  |   1    |   2    |   3    |   4    |
|  H[2]  |   5    |   6    |   7    |   8    |
|  H[1]  |   9    |   0    |   +    |   -    |
|  H[0]  |  &  (  |  |  )  |   <    |   =    |

* Press and release less than 500ms: left operator
* Press and hold longer than 500ms: right operator
